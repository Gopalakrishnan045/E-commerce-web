# E-commerce Client

Simple steps to run and develop the Angular app:

## 1. Install dependencies

```bash
npm install
```

## 2. Start the development server

```bash
ng serve
```
Visit [http://localhost:4200](http://localhost:4200) in your browser.

## 3. Build the project

```bash
ng build
```

## 4. Run unit tests

```bash
ng test
```

## 5. Run end-to-end
